## Что это

Это статическое веб-приложение с листом персонажа:

- поля заполняются прямо в браузере;
- точки можно отмечать кликом;
- данные сохраняются локально в браузере;
- есть светлая и темная тема.

## Файлы проекта

- `index.html` - структура страницы;
- `styles.css` - оформление и темы;
- `script.js` - интерактивность, сохранение и логика точек.
